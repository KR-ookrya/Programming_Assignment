당근마켓 클론 코딩_간단 요약

1.무엇을 만드는 것인가?

- 상품 올리고
- 상품 목록 보고
- 상품 상세보기

2.필요한 기술(간단히)

- React ->화면(UI) 만드는 부분
- Node.js(Express) ->서버(데이터 처리)
- MongoDB ->상품 정보 저장

3. 동작 구조(핵심)

- React(화면) ->Node.js(서버) -> MongoDB(DB)
- React(화면) <-Node.js(서버) <- MongoDB(DB)


4. 최소 기능(초보자 버전)

- 상품 목록 보기
- 상품 등록하기
- 상품 상세 보기

5. 만드는 순서

- React 프로젝트 생성
- Node.js 서버 생성
- MongoDB 연결
- 상품 API 만들기
- React에서 API 가져와 화면에 보여주기


6. 한줄 요약

React로 화면 만들고 ->Node.js로 데이터 처리하고 ->MongoDB에 저장해서 상품을 올리고 보여주는 중고거래 앱을 만드는 연습!
