당근마켓 클론 코딩 : 당근마켓처럼 중고 물건을 올리고 → 목록에 보여주고 → 채팅하거나 구매 요청하는 앱을 비슷하게 만들어 보는 연습이야.
실제 완성품을 만드는 게 목적이 아니라, 웹/앱 개발의 흐름을 이해하는 것이 목표!


무조건 쉽게! 전체 흐름 4단계


1️⃣화면(프론트엔드) 만들기 – React or Flutter : 사용자가 눈으로 보는 화면을 만드는 단계

2️⃣서버(백엔드) 만들기 – Node.js + Express : 웹사이트 뒤에서 데이터를 관리하는 역할

3️⃣ DB(데이터베이스) 연결 – MySQL or MongoDB : 데이터 저장하는 곳

4️⃣ 프론트 + 서버 + DB 연결! 
React에서 API 요청을 보내면: React → 서버(Node) → DB → 서버 → React
