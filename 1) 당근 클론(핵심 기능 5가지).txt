1) 홈 화면( 상품 등록)

- 사람들이 올린 상풀 리스트 보기
- 사진, 제목, 가격, 동네 정보 표시
- 스크롤하면 계속 불러오기(무한 스크롤 가능)

2) 상품 등록 페이지
- 사진 업로드
- 제목/가격/설명 입력
- 등록 버튼 -> DB에 정장

3) 상품 상세 페이지
- 상품 정보 보기
- 판매자 정보 표시
- 채팅하기 버튼

4) 채팅 기능
- 구매자 <-> 판매자 1:1 채팅
- 실시간 기능은 Firebase 같은 서비스를 사용하면 쉬움

5) 동네 기반 기능
- 사용자 위치 가져오기
- 같은 동네의 게시글만 보이도록 필터링
